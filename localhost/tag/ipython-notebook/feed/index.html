There you may select your version of Ubuntu, a server nearby and run the given commands in a command line. I recommend running “sudo echo” before the given commands to unlock sudo first.

After running “sudo apt-get update”, you may install the newest IPython Notebook and start it by running

sudo apt-get install ipython1x-notebook

ipython1x notebook

You will have to use iptyhon1x because NeuroDebian keeps your old IPython installation available as ipython, just in case you regret going forward.
